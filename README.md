## About

I work close to the hardware. My daily driver is **Embedded Linux on ARM Cortex** — building, debugging and shipping software where it actually touches silicon. Alongside that I write **C** and **Rust** for anything performance- or correctness-critical, and **Python** for tooling, testing, and automation.

I also spend time on the infrastructure side: writing and maintaining **Jenkinsfiles** and **GitLab CI** pipelines, building custom Linux images with **Yocto** and flashing them onto target hardware, and keeping my own workflow lean — **Ubuntu**, **Bash** scripts for anything repetitive, **Git**, **Docker**, and an editor setup built around **Neovim + tmux**.

I like reading other people's source code until I actually understand it, not just until it compiles.
